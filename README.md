# AI Resume Analyzer

This is a Streamlit-based web application that analyzes resumes and provides feedback based on selected job roles.

## Features
- Upload PDF resume
- Extract text using pdfplumber
- Detect skills based on job role
- Show missing skills and suggestions
- Calculate resume score
- Recommend best suitable role

## Technologies Used
- Python
- Streamlit
- pdfplumber

## How to Run
1. Install required libraries:
   pip install streamlit pdfplumber

2. Run the app:
   streamlit run app.py
